II - Le recrutement dans les entreprises spécialisées dans l’informatique.

Qu’est ce qu’un éditeur de logiciel ?

Un éditeur de logiciel est une entreprise dont le but est de “ concevoir puis de distribuer un ou plusieurs logiciels ” (1) afin d’être vendu sur un marché tel que Microsoft, IBM, Oracle, ….

A quoi sert un éditeur de logiciel ?

Dans une entreprise éditrice de logiciel, le but est de prévoir les attentes et les demandes des clients et de leur proposer des progiciels adaptés, par exemple Microsoft avec Office ( Word, PowerPoint, Excel, etc ) ou Oracle avec Java et MySql. Dans une entreprise éditrice de logiciel, le but est de prévoir les attentes et les demandes des clients et de leur proposer des progiciels adaptés, par exemple Microsoft avec Office ( Word, PowerPoint, Excel, etc ) ou Oracle avec Java et MySql. Ces entreprises peuvent développer 2 types de logiciel, les “progiciels” dont les exemples cités au dessus font partie, ils doivent donc être conçus pour être utilisables et compréhensibles par tous pour un usage donné (Ex : Tout le monde peut créer une diapositive grâce à PowerPoint, cela ne requiert pas de savoirs particuliers ). Ces logiciels sont généralement accompagnés de documentation pour simplifier l’usage. Le deuxième type de logiciel sont les logiciels spécifiques qui sont destinés à être utilisés dans des entreprises uniquement (Ex : Pronote ou Scodoc peuvent être assimilés à des logiciels spécifiques malgré qu’ils ne soient disponibles qu’en version web). Ils ne sont développés que par des ESN.
