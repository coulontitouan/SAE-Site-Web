Quel est le but de notre site

Ce site web a été réalisé dans le but de vous informer, futurs étudiants en BUT Informatique, sur les entreprises qui recrutent dans le domaine de l’informatique. En effet, vous serez amenés à vous spécialiser dans différents domaines (sécurité, réseaux, développement web…) et il vous sera indispensable de savoir dans lesquels s’orienter.
